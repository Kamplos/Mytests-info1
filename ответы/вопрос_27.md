## 1. Основные задачи анализа данных: прогнозирование, классификация, кластеризация, анализ отклонений

**Анализ данных (Data Analysis)** — это процесс исследования, очистки, преобразования и моделирования данных с целью обнаружения полезной информации, формирования выводов и поддержки принятия решений. Выделяют четыре ключевые задачи:

###  Прогнозирование (Forecasting / Regression)
Предсказание непрерывных числовых значений на основе исторических данных.
*   **Суть:** Поиск закономерностей во временных рядах или зависимостях между переменными для extrapolation в будущее.
*   **Примеры:** Прогноз курса валют,预测 продаж на следующий квартал, оценка стоимости недвижимости, прогноз погоды.
*   **Методы:** Линейная регрессия, ARIMA, нейронные сети (LSTM).

### 🏷️ Классификация (Classification)
Отнесение объектов к заранее определённым категориям (классам).
*   **Суть:** Обучение модели на размеченных данных (supervised learning), где каждому объекту присвоен правильный ответ. Модель learns boundaries between classes.
*   **Примеры:** Спам-фильтр (спам/не спам), диагностика заболеваний (болен/здоров), распознавание рукописных цифр, кредитный скоринг.
*   **Методы:** Decision Trees, Random Forest, SVM, Logistic Regression, Neural Networks.

### 🧩 Кластеризация (Clustering)
Группировка объектов по схожести без заранее известных меток.
*   **Суть:** Unsupervised learning. Алгоритм самостоятельно находит структуру в данных, объединяя похожие элементы в группы (кластеры).
*   **Примеры:** Сегментация клиентов (для таргетированной рекламы), группировка новостей по темам, выявление типов поведения пользователей, биологическая таксономия.
*   **Методы:** K-Means, DBSCAN, Hierarchical Clustering.

### ⚠️ Анализ отклонений (Anomaly Detection)
Выявление объектов, событий или наблюдений, которые существенно отличаются от большинства данных.
*   **Суть:** Поиск "иголок в стоге сена". Отклонения могут указывать на ошибки, мошенничество или редкие важные события.
*   **Примеры:** Обнаружение мошеннических транзакций в банке, поиск кибератак в сетевом трафике, контроль качества на производстве (брак), мониторинг состояния оборудования (предсказание поломок).
*   **Методы:** Isolation Forest, One-Class SVM, Statistical Z-score, Autoencoders.

## 2. Задача на поиск кратчайшего пути в графе

### 📋 Условие
Дана таблица расстояний между пунктами A, B, C, D, E, F. Необходимо найти длину кратчайшего пути из пункта **A** в пункт **F**.

**Таблица смежности (веса рёбер):**
*   A-B: 3
*   A-C: 5
*   A-F: 15
*   B-D: 4
*   C-D: 1
*   D-E: 2
*   D-F: 6
*   E-F: 1

###  Решение (Алгоритм Дейкстры / Перебор путей)

Будем последовательно находить кратчайшие расстояния от起点 A до всех остальных вершин.

1.  **Старт в точке A:**
    *   Расстояние до A = 0.
    *   Соседи A: B (3), C (5), F (15).
    *   Текущие мин. расстояния: $d(B)=3$, $d(C)=5$, $d(F)=15$.

2.  **Переход в точку B** (так как 3 < 5 < 15):
    *   Из B можно попасть в D.
    *   Путь A → B → D: $3 + 4 = 7$.
    *   Сравним с текущим знанием о D (пока $\infty$). $7 < \infty$, значит $d(D) = 7$.

3.  **Переход в точку C** (следующая ближайшая непосещённая вершина, $d(C)=5$):
    *   Из C можно попасть в D.
    *   Путь A → C → D: $5 + 1 = 6$.
    *   Сравним с текущим $d(D)=7$. Так как $6 < 7$, обновляем кратчайший путь до D: **$d(D) = 6$**.
    *   *(Маршрут стал: A → C → D)*

4.  **Переход в точку D** (теперь $d(D)=6$, это меньше чем прямое A→F=15):
    *   Из D можно попасть в E и F.
    *   **Путь в E:** A → ... → D → E = $6 + 2 = 8$. Значит, $d(E) = 8$.
    *   **Путь в F (через D):** A → ... → D → F = $6 + 6 = 12$.
        *   Сравним с прямым путём A → F = 15.
        *   $12 < 15$, значит, текущий рекорд до F: **$d(F) = 12$**.

5.  **Переход в точку E** ($d(E)=8$):
    *   Из E можно попасть в F.
    *   **Путь в F (через E):** A → ... → D → E → F = $8 + 1 = 9$.
    *   Сравним с текущим рекордом $d(F)=12$.
    *   $9 < 12$, значит, обновляем кратчайший путь: **$d(F) = 9$**.

6.  **Финальная проверка:**
    *   Осталась только вершина F. Все пути рассмотрены.
    *   Кратчайший маршрут: **A → C → D → E → F**.
    *   Длина: $5 + 1 + 2 + 1 = 9$.

### ✅ Ответ
> **9**
