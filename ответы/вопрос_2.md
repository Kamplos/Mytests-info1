# Билет №2

## 1. Программное обеспечение компьютера. Особенности ПО мобильных устройств

### 💻 Программное обеспечение (ПО) компьютера
**Программное обеспечение (Software)** — это совокупность программ, которые управляют работой аппаратных средств компьютера, а также обеспечивают взаимодействие с пользователем.

ПО традиционно делится на три основные категории:

1. **Системное ПО**
   * **Назначение:** Обеспечивает работу самого компьютера, управление его ресурсами и создает среду для выполнения других программ.
   * **Примеры:** Операционные системы (Windows, Linux, macOS), драйверы устройств, системные утилиты (антивирусы, дефрагментаторы, архиваторы).
2. **Прикладное ПО**
   * **Назначение:** Предназначено для решения конкретных пользовательских задач в различных сферах деятельности.
   * **Примеры:** Текстовые редакторы (Word), браузеры (Chrome), графические редакторы (Photoshop), игры, бухгалтерские системы.
3. **Инструментальное ПО (Системы программирования)**
   * **Назначение:** Используется для создания, тестирования и отладки нового программного обеспечения.
   * **Примеры:** Компиляторы, интерпретаторы, интегрированные среды разработки (IDE), такие как Visual Studio, PyCharm.

---

### 📱 Особенности ПО мобильных устройств
Программное обеспечение для смартфонов и планшетов имеет ряд принципиальных отличий от классического ПК:

* **Ограниченность ресурсов:** Мобильные ОС жестко ограничивают потребление оперативной памяти и процессорного времени фоновыми приложениями для экономии заряда батареи.
* **Специфический ввод/вывод:** ПО оптимизировано под сенсорные экраны (жесты, мультитач) и учитывает наличие специфических датчиков (акселерометр, гироскоп, GPS, NFC, сканер отпечатков).
* **Централизованное распространение:** В отличие от ПК, где программы можно скачать откуда угодно, мобильное ПО чаще всего распространяется через официальные магазины приложений (App Store, Google Play), что обеспечивает строгий контроль безопасности и качества.
* **Архитектура процессоров:** Мобильное ПО компилируется под ARM-архитектуру (в отличие от классической x86/x64 на ПК), что требует использования энергоэффективных инструкций.
* **Модель «песочницы» (Sandbox):** Каждое мобильное приложение работает в изолированной среде и не имеет прямого доступа к файлам других приложений или системным файлам без явного разрешения пользователя.

## 2. Задача на неравномерный двоичный код (Код Фано)

### 📋 Условие
* Алфавит: **И, Б, О, К, Л** (5 букв).
* Условие Фано: ни одно кодовое слово не является началом другого (префиксный код).
* Известно: **И = `0`**, **Л = `100`**.
* Найти: **наименьшую возможную длину** закодированного слова **ОБЛИК**.

### 🧮 Решение

**Шаг 1. Анализ известных кодов и построение дерева**
* Код буквы **И** — `0`. Это означает, что все остальные коды должны начинаться с `1` (иначе `0` будет префиксом).
* Код буквы **Л** — `100`. 

Посмотрим на доступные ветви в дереве, начинающиеся с `1`:
* Ветвь `10`:
  * `100` — занята буквой **Л**.
  * `101` — **свободна** (длина 3).
* Ветвь `11`:
  * `11` — **свободна** (длина 2).
  * `110` — **свободна** (длина 3).
  * `111` — **свободна** (длина 3).

**Шаг 2. Подбор кодов для оставшихся букв (О, Б, К)**
Нам нужно закодировать 3 буквы: **О, Б, К**. Чтобы длина слова была минимальной, нужно выбрать самые короткие доступные коды.

* *Можно ли использовать код `11` (длина 2)?* 
  Если мы назначим букве `11`, то согласно условию Фано, коды `110` и `111` станут недоступны (так как `11` будет их началом). В этом случае у нас останется только один свободный код — `101`. Но нам нужно закодировать **три** буквы! Значит, использовать `11` **нельзя**.
  
* *Значит, мы вынуждены использовать коды длины 3:*
  Доступные свободные коды длины 3: **`101`**, **`110`**, **`111`**.
  Их как раз хватает для трёх оставшихся букв (О, Б, К).

**Шаг 3. Вычисление длины слова**
Распределим коды (порядок не важен, так как длины одинаковые):
* **О** = `101` (длина 3)
* **Б** = `110` (длина 3)
* **Л** = `100` (длина 3) *(дано)*
* **И** = `0` (длина 1) *(дано)*
* **К** = `111` (длина 3)

Теперь посчитаем суммарную длину слова **ОБЛИК**:
$$L = L(О) + L(Б) + L(Л) + L(И) + L(К)$$
$$L = 3 + 3 + 3 + 1 + 3 = 13$$

### ✅ Ответ
> **Наименьшая возможная длина слова ОБЛИК равна 13.**
