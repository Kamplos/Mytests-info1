## 1. Мультимедиа. Использование мультимедийных онлайн-сервисов для разработки презентаций проектных работ. Виды и форматы мультимедийных файлов

### 🎬 Понятие мультимедиа
**Мультимедиа** (от лат. *multum* — много, *medium* — средство) — это технология, позволяющая одновременно использовать в информационном продукте текст, графику, звук, видео и анимацию. Ключевая особенность современной мультимедиа — **интерактивность**, то есть возможность взаимодействия пользователя с контентом.

### ☁️ Онлайн-сервисы для создания презентаций
Традиционные десктопные программы (PowerPoint, LibreOffice Impress) дополняются облачными сервисами, которые упрощают совместную работу над проектными заданиями:
*   **Google Презентации / Яндекс.Телемост:** Бесплатный доступ, автоматическое сохранение, комментирование в реальном времени, шаблоны.
*   **Canva / Figma Slides:** Огромная библиотека дизайна, стоковых фото/видео, интуитивный drag-and-drop интерфейс, адаптация под соцсети.
*   **Prezi:** Нелинейная структура презентации (зум-эффект), динамичная навигация вместо слайдов.
*   **Gamma / Beautiful.ai:** Сервисы на базе ИИ, генерирующие структуру и дизайн слайдов по текстовому описанию.

**Преимущества для проектной работы:**
1.  Доступ к проекту с любого устройства.
2.  История версий и возможность отката изменений.
3.  Встраивание медиафайлов прямо из интернета без загрузки на диск.
4.  Экспорт в PDF, PPTX или публикация по ссылке.

### 📂 Виды и форматы мультимедийных файлов

| Тип контента | Популярные форматы | Особенности |
| :--- | :--- | :--- |
| **Растровая графика** | JPEG, PNG, GIF, WebP | JPEG — фото с потерями; PNG — прозрачность, без потерь; GIF — простая анимация; WebP — современный универсальный формат |
| **Векторная графика** | SVG, AI, CDR | Масштабируется без потери качества, используется для схем и логотипов |
| **Аудио** | MP3, WAV, FLAC, AAC | MP3/AAC — сжатие с потерями (стриминг); WAV/FLAC — без потерь (архивное качество) |
| **Видео** | MP4 (H.264/H.265), WebM, AVI | MP4 — универсальная совместимость; WebM — оптимизирован для веба; AVI — контейнер без жесткого стандарта сжатия |
| **Анимация** | Lottie (JSON), APNG | Lottie — векторная анимация для веб/приложений; APNG — улучшенная альтернатива GIF |

> 💡 **Совет:** При подготовке проектной презентации используйте формат **PDF** для финальной сдачи (гарантия идентичного отображения) и **PPTX/онлайн-ссылку** для защиты (сохранение анимации и видео).

## 2. Задача на маски имён файлов

### 📋 Условие
Маска: `*?tos?*.??*`

Правила:
*   `?` — ровно один произвольный символ.
*   `*` — любая последовательность символов произвольной длины (включая пустую).

Варианты:
1.  `tosman.doc`
2.  `etosd.e`
3.  `utos.cp`
4.  `rtosh.ben`

### 🧮 Решение

Разберём маску на обязательные элементы:
`*` + `?` + `tos` + `?` + `*` + `.` + `?` + `?` + `*`

Это означает:
1.  Перед подстрокой `tos` должен быть **минимум один** символ (обязательный `?`).
2.  После подстроки `tos` должен быть **минимум один** символ (обязательный `?`), а затем может быть что угодно (`*`).
3.  Расширение файла должно содержать **минимум два** символа (два обязательных `?` после точки).

Проверим каждый вариант:

| № | Файл | Проверка имени | Проверка расширения | Результат |
|---|---|---|---|---|
| 1 | `tosman.doc` | `tos` стоит в самом начале → перед ним **0** символов, а нужен минимум 1 (`?`) | `.doc` → 3 символа ≥ 2 ✅ | ❌ Не подходит |
| 2 | `etosd.e` | `e` + `tos` + `d` → `?`=e, `tos`, `?`=d, `*`=пусто ✅ | `.e` → **1** символ, а нужно минимум 2 (`??`) | ❌ Не подходит |
| 3 | `utos.cp` | `u` + `tos` + *(нет символа после tos)* → `?` после `tos` требует хотя бы один символ, но сразу идёт точка | — | ❌ Не подходит |
| 4 | `rtosh.ben` | `r` + `tos` + `h` → `?`=r, `tos`, `?`=h, `*`=пусто ✅ | `.ben` → 3 символа ≥ 2 ✅ | ✅ **Подходит** |

Подробная проверка варианта 4 (`rtosh.ben`):
*   `*` → пустая последовательность
*   `?` → `r`
*   `tos` → `tos`
*   `?` → `h`
*   `*` → пустая последовательность
*   `.` → `.`
*   `?` → `b`
*   `?` → `e`
*   `*` → `n`

Все условия маски выполнены.

### ✅ Ответ
> **4) rtosh.ben**
