## 1. Алгоритмы. Виды и структуры алгоритма

### 🧠 Понятие алгоритма
**Алгоритм** — это конечная последовательность четко определенных инструкций (шагов), выполнение которых приводит к решению поставленной задачи за конечное время.

**Основные свойства алгоритма:**
1.  **Дискретность:** Процесс решения разбит на отдельные элементарные шаги.
2.  **Определенность (детерминированность):** Каждый шаг должен быть однозначным и понятным исполнителю; не должно быть двусмысленности.
3.  **Конечность (финитность):** Алгоритм должен завершаться за конечное число шагов.
4.  **Результативность:** После завершения работы должен быть получен конкретный результат.
5.  **Массовость:** Алгоритм должен быть применим к целому классу однотипных задач, а не только к одному частному случаю.

### 🔀 Базовые структуры алгоритмов
Любой сложный алгоритм можно составить из трех базовых конструкций:

1.  **Следование (Линейная структура):** Инструкции выполняются строго последовательно, одна за другой.
2.  **Ветвление (Развилка):** Выбор дальнейшего пути зависит от проверки условия. Бывает полным (если... то... иначе) и неполным (если... то).
3.  **Цикл (Повторение):** Многократное выполнение действий.
    *   *С предусловием (WHILE):* Проверка перед действием.
    *   *С постусловием (DO-WHILE):* Проверка после действия.
    *   *С параметром (FOR):* Количество повторений известно заранее.

## 2. Логическое выражение: Схема и таблица истинности

###  Исходное выражение
$$ F = \neg(A \lor B) \land (A \land \neg B) $$

> ️ **Анализ логики:**
> Первая часть $\neg(A \lor B)$ истинна только при $A=0, B=0$.
> Вторая часть $(A \land \neg B)$ истинна только при $A=1, B=0$.
> Эти условия противоречат друг другу ($A$ не может быть одновременно 0 и 1). Следовательно, выражение является **тождественно ложным**.

### 🔌 Электронная логическая схема

Для реализации схемы потребуются:
*   1 элемент **ИЛИ** (OR)
*   2 элемента **НЕ** (NOT)
*   2 элемента **И** (AND)

**Схема соединений:**
```text
      ┌──────────┐
A ────┤          │
      │   OR     ├───[NOT]──────────────┐
B ────┤          │                      │
      └──────────┘                      │
                                        ├────[AND]──── F (Выход = 0)
      ┌──────────┐                      │
A ────────────────┤          │          │
                  │   AND    ├──────────
B ────[NOT]───────┤          │
                  └──────────┘
```
